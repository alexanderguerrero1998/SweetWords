# SweetWords
Es una aplicación móvil personalizada desarrollada en Android con el leguaje Java </br> 
<h4>Caracteristicas dela app</h4>
<ul>
  <li>Al iniciar la aplicacion se reproduce una melodia</li>
  <li>Al tocar la pantalla permite ver diferentes frases preestablecidas </li>
</ul>
